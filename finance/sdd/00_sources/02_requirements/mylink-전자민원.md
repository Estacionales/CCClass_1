[요구 3-1] 사용자가 마이데이터 동의를 완료한 상태에서
  전자민원 발급을 요청하면, 시스템은 자격 규칙에 따라
  필요한 서류 목록을 산출하고 해당 연계기관에서 수집한다.

[요구 3-2] 연계기관 응답이 3초 내 도착하지 않으면,
  회복력 정책(재시도 N회 → 서킷브레이커 → 대체 경로)으로
  우아하게 저하(graceful degradation)한다.

[요구 4-1·근거 조회] '이 민원 → 필요 서류 → 근거 규정 → 예외'의
  관계를 빠짐없이 따라가 근거 규정까지 인용한다.

[요구 4-2·멱등] 발급·정산 배치가 재실행돼도 중복 발급/정산을 만들지 않는다.

[요구 4-3·동의 철회] 사용자가 동의를 철회하면 수집 데이터 처리를 중단·파기하고
  동의 원장에 기록한다.
