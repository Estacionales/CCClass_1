# 방명록: 요구사항 원문 (강의 데모용 가상)

> 실재 개인정보 없음.

- 방문자는 다른 사용자의 미니홈피 방명록에 메시지를 남길 수 있다.
- 방명록 메시지는 작성 시각 순서로 조회된다.
- 방명록 주인 또는 메시지 작성자 본인은 해당 메시지를 삭제할 수 있다.
- 주인도 작성자도 아닌 사용자는 메시지를 삭제할 수 없다.
- 비밀글로 작성된 메시지는 주인과 작성자 본인만 본문을 볼 수 있고, 그 외 사용자에게는 마스킹되어 보인다.
- 방명록 주인은 특정 메시지에 답글을 남길 수 있으며, 메시지당 답글은 1개까지만 허용된다.
- 이미 답글이 달린 메시지에 추가 답글을 시도하면 거부된다.
