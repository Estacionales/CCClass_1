# 미니홈피: 요구사항 원문 (강의 데모용 가상)

> 실재 개인정보 없음. 이미지·BGM 등 콘텐츠는 예시 메타데이터로만 다룬다.

- 회원가입이 완료되면 해당 사용자 소유의 미니홈피가 자동 생성된다.
- 미니홈피는 프로필(닉네임·상태메시지·스킨), 다이어리(게시글), 사진첩으로 구성된다.
- 다이어리 게시글은 작성 시 공개범위(전체공개·일촌공개·비공개)를 지정한다.
- 비공개 게시글은 작성자 본인만 조회할 수 있다.
- 일촌공개 게시글은 작성자와 일촌 관계인 사용자만 조회할 수 있다.
- 본인이 아닌 사용자가 미니홈피를 방문하면 방문자수(파도타기 카운트)가 1 증가한다.
- 같은 방문자의 24시간 내 재방문은 방문자수를 중복 증가시키지 않는다.
- 스킨·BGM은 도토리로 구매해 인벤토리에 보유한 아이템만 적용할 수 있다.
- 보유하지 않은 스킨·BGM을 적용하려는 요청은 거부된다.
