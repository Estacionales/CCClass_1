# 일촌(一寸): 요구사항 원문 (강의 데모용 가상)

> 실재 개인정보·실명 없음. 사용자 식별자는 불투명 예시값(`user-001` 등)만 사용.

- 사용자는 다른 사용자에게 일촌 요청을 보낼 수 있다.
- 상대가 수락하면 양방향 일촌 관계가 성립한다.
- 이미 일촌이거나 대기 중(PENDING) 요청이 있는 대상에게 재요청해도 중복 생성되지 않는다.
- 일촌 요청은 발급 후 24시간 내에 응답이 없으면 만료(EXPIRED)된다.
- 만료된 요청 이후에는 재요청이 가능하다.
- 사용자 1인당 최대 일촌 수는 750명이다(원조 싸이월드 정원 유지).
- 정원이 찬 상태에서 새 요청을 수락하면 정원 초과로 거부된다.
- 일촌을 해제(파일촌)하면 양쪽 사용자 목록에서 동시에 관계가 사라진다.
- 일촌평(한 줄 소개)은 일촌 관계가 성립한 사용자 사이에서만 작성할 수 있다.
